# 🅿️ Smart Parking System
A full-stack parking management solution featuring **Gemini AI** for dynamic greetings and a modern glassmorphism UI.

## 🚀 Features
* **AI Greeting:** Uses Google Gemini to generate witty, time-aware welcomes for drivers.
* **Real-time Dashboard:** Tracks occupancy, vehicle types (Car/Bike), and daily stats.
* **Automated Billing:** Calculates costs based on duration and vehicle type.
* **Admin Panel:** Manage slots, view history, and generate printable PDF receipts.
* **Dark Mode:** Fully responsive UI with light/dark theme toggle.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **AI:** Google Generative AI (Gemini Flash)

## 📦 Installation
1. Clone the repo: `git clone https://github.com/YourUsername/smart-parking-system.git`
2. Install dependencies: `npm install`
3. Create a `.env` file and add `GEMINI_API_KEY=your_key_here`
4. Start the server: `node server.js`
